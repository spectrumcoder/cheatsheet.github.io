HTML

HTML consists of elements, structure, and tables.

HTML Elements

Tag Name Description
<body> body the body element contains the content of an HTML document
<div> div the div element acts as a containter to divide the HTML document into sections
<h1..h6> heading HTML can use six different levels of heading elements
<p> paragraph the paragraph element contains and displays a block of text
<span> span the span element groups inline text for the purposes of styling text
<strong> strong the strong element is used to highlight important text and will normally be bolded
<em> emphasis the emphasis element will emphasis text by italicizing the emphasised text
<br> line break The line break element will create a line break in the text
<ol> ordered list the ordered list element will create a list of items in sequential order
<ul> unordered list the unordered list element will create a list of items in no particular order
<li> list item the list item element holds list items
<video> video the video element embeds a media player for video playback
controls controls the controls attribute will display video controls in the media player
<img> image the image elements embeds images in documents
src src the source attribute will contain a url
alt alt the alt attribute contains alternative text if a url is not available
id id the id attributes can be assigned to different elements in order to differntiate between them

HTML Structure

Tag Name Description
<html> html all content of an HTML document shoudl be contained between the opening and closing HTML tags
<head> head the head element will contain general information about the HTML page
<title> title the title element is contained inside the head element and contains the title of the HTML page
<a> anchor the a anchor element is used to create a hyperlink in the HTML document
href href the href determines the the location othe anchor element points to
target target the target attribute specifies where a hyperlink should be opened
file path file path HTML file paths can be absoulte paths or relative paths

HTML Tables

Tag Name Description
<table> table the table element contains all table contents
<thead> table head the table head element defines the heading of the table
<tbody> table body the table body element contains all table data
<tfoot> table footer the table footer element summarizes content at the end of the table
<th> table heading the table heading element is used to add titles to rows or columns
<tr> table row the table row element is used to add rows to a table
<td> table data the table data element adds data to a cell of a table
colspan colspan attribute defines how many columns a td element should span
rowspan rowspan attribute defines how many rows a td element should span
